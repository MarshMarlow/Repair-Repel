# Repair-Repel
video game

Helmet Lantern:
- Equip: press 'Q' ewhile facing it
- Unequip: press 'Q' while facing the table it was on (tracks this table with a layer)
- Turn on/off: press 'F' while helmet is equipped

Work Table (For fizing helmet lantern):
- Recharge helmet: unequip helmet to table (press 'Q' while facing the table) then leave it as it recharges over time

Anvil (For fixing weapons)
1. Place weapon anvil through object grabbable controls (mouse 0)
2. Grab hammer (mouse 0) that is on the anvil then hit the weapon that is on the table
3. Continue until durability increases to amount you want

- Weapon is identified with 'Weapon' tag
- Crossbow is placed on the anvil for testing
